# 功能特性

## 核心功能

### 📄 PDF 上传
- 支持拖拽上传
- 支持点击选择文件
- 单文件最大 250MB
- 仅支持 PDF 格式

### 🎯 目标大小压缩
- 输入期望的文件大小（MB）
- 智能算法自动寻找最佳压缩参数
- 优先保证图片和文字清晰度
- 保持页数、页面尺寸和版式不变

### 📊 实时进度
- 显示压缩进度百分比
- 实时状态更新
- 压缩前后文件大小对比
- 压缩比例显示

### 💾 一键下载
- 压缩完成自动提供下载
- 文件名自动添加 `.compressed` 后缀
- 支持批量下载（单次任务）

## 技术特性

### macOS 原生压缩
- 使用 PDFKit 和 QuartzFilter 框架
- 系统级优化，压缩质量最佳
- 支持光栅化 fallback 模式

### 智能压缩策略
1. 首先尝试无损和轻度压缩
2. 逐步降低图片质量和缩放
3. 最后尝试光栅化处理
4. 始终在目标大小内寻找最佳质量

### 隐私安全
- 所有处理在本地完成
- 文件不上传任何第三方服务器
- 临时文件自动清理
- 无用户数据收集
